Instructions

  • Register an account if you haven't already
  • Login to your account after creating one
  • Create a new snippet
  • Make sure to add file extension to the title of snippet. Ex: program.py, example.java, hello_world.c
  • Don't worry if you didn't add an extension. We will try and guess your language too!
  • Add a tag if you wish. Make sure to use the form #tag or even add multiple tags to a snippet: #tag1, #tag2
  • Then you can view your snippet in the dashboard
  • Use the search bar to search by title, file extension, or even tag!
  • To search by tag enter # then the name of the tag: #tag1
  • Edit your snippet too. Tags, titles, and content can all be edited
  • Delete and copy your code using the corresponding buttons as well.
